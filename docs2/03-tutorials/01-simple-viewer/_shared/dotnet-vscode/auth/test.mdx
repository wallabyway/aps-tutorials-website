If the application is still running, restart it (for example, using _Run > Restart Debugging_,
or by clicking the green restart icon), otherwise start it again (using _Run > Start Debugging_,
or by pressing `F5`). When you navigate to [http://localhost:8080/api/auth/token](http://localhost:8080/api/auth/token)
in the browser, the server should now respond with a JSON object containing the access token data.
