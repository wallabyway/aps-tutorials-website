For the coding itself you're welcome to use whichever editor you like the most.
If you're looking for recommendations, we recommend [Visual Studio Code](https://code.visualstudio.com)
as it is free, cross-platform, fast, and packed with lots of great features.
Simply install the latest stable build for your platform.

:::tip
We've built an extension for Visual Studio Code that provides insight into
some of the APS services and data directly from the editor:
[Autodesk Forge Tools](https://marketplace.visualstudio.com/items?itemName=petrbroz.vscode-forge-tools).
This could be useful when debugging your own APS applications.
:::


Then you need to install [C# Extension](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp) (if don't have it yet) for Visual Studio Code. To install, click on extensions tab on the left sidebar of your Visual Studio Code, then search for ```C#``` and click install as shown in the image below.

:::info
This extension is used for developing web applications in C# in the Visual Studio Code editor:
[C# Extension](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp).
This could be useful when setting up and debugging your own APS applications.
:::
